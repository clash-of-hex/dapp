## Develop

```shell
npm install
```

## Develop App

```shell
cd src/app
npm run se-reset
npm test
```

## Develop Web

```shell
npm run build-app
cd src/web
npm start
```

## Preview

```shell
npm start
```

## Deploy

```shell
git remote add gh-pages git@github.com:clash-of-hex/clash-of-hex.github.io.git
npm run deploy
```
