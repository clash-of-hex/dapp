# Clash of Hex

## Develop

```shell
npm install
```

## Develop Web

```shell
npm run build-app
cd src/web
npm start
```

## Preview

```shell
npm start
```

## Deploy

```shell
git remote add gh-pages git@github.com:clash-of-hex/clash-of-hex.github.io.git
npm run deploy
```
